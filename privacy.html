プライバシーポリシー

制定日：2025年2月22日 ／ 最終改定日：2026年4月20日

第1条（総則）

segawa takeshi（以下「提供者」といいます）は、iOSアプリケーション「ヤメッター」（以下「本アプリ」といいます）において、ユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

第2条（収集する情報）

提供者は、本アプリの利用にあたり、以下の情報を収集することがあります。

  1. Appleでサインインを通じて取得するアカウント情報（ユーザー識別子、メールアドレス等）
  2. ユーザーが設定するプロフィール情報（ニックネーム、アイコン画像、地域、性別、自己紹介、学習目標等）
  3. ユーザーが投稿するテキスト・コメント・集中タイムの記録（時間グラフ・タグ等）
  4. ソーシャル機能に関する情報（フォロー・フォロワー関係、いいね、ブロック対象等）
  5. マッチング機能に関する情報（セッション参加状態、パートナーとの接続情報、ハートビートデータ等）
  6. プッシュ通知の送受信に必要なトークン情報（FCMトークン）
  7. 集中タイムのセッション記録（開始・終了時刻、設定時間、経過時間、タグ、パートナー情報、達成実績等）
  8. 本アプリの利用状況に関するログ情報（アクセス日時、使用機能等）
  9. 端末情報（機種名、OSバージョン、アプリバージョン等）
  10. エラーログ情報（エラー内容、発生箇所、端末情報等。アプリの品質改善のため自動送信されます）
  11. 広告配信のために必要な識別情報（広告ID等。AppTrackingTransparencyにより許可を求めます）

第3条（情報の利用目的）

収集した情報は以下の目的で利用します。

  1. 本アプリのサービス提供・運営・改善
  2. マッチング機能の提供（他ユーザーとのペアリング、リアルタイム状態共有）
  3. ソーシャル機能の提供（投稿表示、フォロー関係管理、通知配信）
  4. ユーザーサポートへの対応
  5. 広告の配信および最適化
  6. 不正利用・規約違反の検知および対応（NGワードフィルター等）
  7. アプリの品質改善（エラーログの分析）
  8. 利用状況の分析および新機能の開発

第4条（第三者への情報提供）

提供者は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命・身体・財産の保護のために必要な場合
  4. 公衆衛生の向上または児童の健全な育成のために特に必要な場合
  5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

第5条（広告について）

本アプリはGoogle LLCが提供するGoogle AdMobを使用して広告を表示しています。Google AdMobは、広告配信の最適化のためにデバイス識別子（広告ID等）その他の情報を収集することがあります。収集される情報および利用目的については、GoogleのプライバシーポリシーをGoogleのウェブサイト（https://policies.google.com/privacy）にてご確認ください。

ユーザーはデバイスの設定から広告IDをリセットしたり、パーソナライズド広告をオプトアウトしたりすることができます。

第6条（Firebase について）

本アプリはGoogle LLCが提供するFirebaseを使用しています。利用しているFirebaseサービスは以下のとおりです。

  1. Firebase Authentication：ユーザー認証
  2. Cloud Firestore：ユーザープロフィール、投稿、フォロー関係、セッション記録、エラーログ等のデータ保存
  3. Firebase Realtime Database：マッチング機能におけるリアルタイム状態同期
  4. Firebase Storage：ユーザーアイコン画像の保存
  5. Firebase Cloud Messaging（FCM）：プッシュ通知の送受信
  6. Firebase Remote Config：アプリ設定・ヘルプテキストの配信
  7. Firebase Analytics：アプリ利用状況の分析

Firebaseが収集する情報および利用目的については、GoogleのプライバシーポリシーをGoogleのウェブサイト（https://policies.google.com/privacy）にてご確認ください。

第7条（Appleのサービスについて）

本アプリはApple Inc.が提供する以下のサービスを使用しています。

  1. Appleでサインイン（Sign in with Apple）：ユーザー認証
  2. Screen Time API / FamilyControls：集中タイム中のアプリ制限機能（ユーザーの承認を得た場合のみ）
  3. Apple Push Notification service（APNs）：プッシュ通知の配信基盤

Appleによる情報の取り扱いについては、Apple Inc.のプライバシーポリシー（https://www.apple.com/jp/legal/privacy/）をご確認ください。

第8条（外部アプリとのデータ共有）

本アプリは、提供者が提供する別アプリ「スマやめ」（Fish2.0）との間で、App Group機能を通じて以下のデータを共有する場合があります（ユーザーが設定で連携を有効化した場合のみ）。

  1. 集中タイムのセッションデータ（開始・終了時刻、経過時間等）
  2. アプリ制限設定（ブロック対象アプリの選択情報）

このデータ共有はデバイス内で完結し、外部サーバーを経由しません。

第9条（データの保存場所）

本アプリは、ユーザーのデータを以下の場所に保存します。

  1. デバイス内（Keychain）：プロフィール、セッション履歴、設定情報等を暗号化して保存します。アプリ削除時に自動的に削除されます。
  2. クラウド（Firebase）：プロフィール、投稿、フォロー関係、セッション記録等をFirebaseサーバーに保存します。データはGoogle Cloud Platformのインフラ上で管理されます。

第10条（情報の管理）

提供者は、収集した個人情報の漏洩・紛失・改ざん等を防ぐため、適切な安全管理措置を講じます。ただし、インターネット上での完全な安全性を保証するものではありません。

第11条（情報の開示・訂正・削除）

ユーザーは、提供者が保有する自己の個人情報について、開示・訂正・削除を求めることができます。ご希望の場合は、本ポリシー末尾の連絡先までお問い合わせください。

アカウントの削除は、本アプリの設定画面から申請できます。削除時、プロフィール情報、投稿・コメント、フォロー関係、セッション記録、プッシュ通知用トークン、アイコン画像がサーバーから削除されます。削除後のデータ復元はできません。

第12条（未成年者のプライバシー）

本アプリは13歳未満の方を対象としていません。13歳未満の方から意図せず個人情報を収集していることが判明した場合、速やかに当該情報を削除します。

第13条（ポリシーの変更）

提供者は、必要に応じて本ポリシーを変更することがあります。変更後のポリシーは本アプリ内または本ページにて通知します。重要な変更がある場合は、アプリ内での通知等により告知します。
